# venta express

**El proyecto consiste en desarrollar un sistema de gestión para la tienda AutoParts Express que permita digitalizar el control de inventario, las ventas, la facturación y la generación de reportes, con el fin de mejorar la atención al cliente y optimizar la operación del negocio**

1. Clonar el repositorio del proyecto.
2. Instalar las dependencias necesarias.
3. Configurar la base de datos del sistema.
4. Ejecutar la aplicación desde el entorno de desarrollo.
5. Acceder al sistema con un usuario autorizado.
